# Contributing

Thanks for helping improve this PG Materials resource hub! 🙌

1. Open an **Issue** or **Pull Request** editing the relevant `courses/<topic>.md`.
2. Resources must be **free** (open or audit-free) and graduate/research quality.
3. Use the table format: `Resource | Provider | Format | Level | Notes`.
4. Keep the README a curated index; put deep lists in `courses/`.

Legend: 🎥 video · 📄 notes/PDF · 📝 problem sets · 🧪 labs/sims · 🆓 fully free.
