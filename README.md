Advanced **free** resources for **M.Sc. / M.Tech. Materials Science** — deep dives into **characterization**, **thin films/surface engineering**, **polymers & nanocomposites**, **battery/energy storage**, **phase transformations**, and **computational (DFT/MD)**. This README is a curated index; deep lists live in [`courses/`](courses/).

## 📖 How to use this repo
1. Skim the curated tables, then open deep-dive files in [`courses/`](courses/).
2. Build computational skills via [`tools/software-and-simulation.md`](tools/software-and-simulation.md).
3. Follow the [roadmap](roadmap.md): characterization → computation → battery R&D.
