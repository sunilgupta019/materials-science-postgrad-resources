# Computational Materials (DFT / MD)

> First serious computational workflows: DFT, MD, and materials databases.

**Prerequisites:** Python, solid state physics, thermodynamics.

## ⭐ Top Picks
- 🎥 **Quantum ESPRESSO Online Course** — https://www.quantum-espresso.org/learn
- 🧪 **LAMMPS tutorials** — https://www.lammps.org/tutorials.html
- 🎥 **MIT 3.320 Atomistic Computer Modeling** — https://ocw.mit.edu

## DFT codes
| Tool | Description | Free Course |
|---|---|---|
| Quantum ESPRESSO | Plane-wave DFT | QE Online Course · PARADIM (JHU) |
| VASP | Industry-standard DFT | VASP Wiki · MIT 3.320 |
| GPAW | Real-space DFT (PAW) | https://gpaw.readthedocs.io · CAMD/DTU YouTube |
| ABINIT | DFT + DFPT | https://docs.abinit.org/tutorial |

## Molecular dynamics
| Tool | Use | Course |
